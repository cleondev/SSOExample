Resource API · .NET 10

API backend cho jQuery SPA

Validate access token, expose endpoint nghiệp vụ, ghi audit log. Project này không host UI người dùng cuối — mở Web SPA để thử flow đầy đủ.

Provider — Authority — Audience — Scope —
Auth

POST /api/auth/login/password

Login bằng username/password — chỉ phục vụ local demo, production redirect sang Microsoft Entra ID.

SSO

GET /api/sso/authorize

Bắt đầu Authorization Code + PKCE. Trả về code và state.

SSO

POST /api/sso/token

Đổi code + code_verifier lấy access token và refresh token.

Auth

POST /api/auth/token/refresh

Refresh access token. Refresh token rotation: token cũ bị xoá sau khi đổi.

Admin

POST /api/auth/login-as

Backdoor có kiểm soát. Bắt buộc role Admin + reason ≥ 10 ký tự. Audit log immutable.

Business

GET /api/me

Trả về principal hiện tại theo bearer token.

Business

GET /api/orders

Endpoint nghiệp vụ ví dụ. Cần bearer token hợp lệ.

Admin

GET /api/admin/users

Danh sách user. Chỉ Admin truy cập.

Admin

GET /api/admin/audit-logs

Audit log: password_login, sso_authorize, login_as. Chỉ Admin truy cập.
